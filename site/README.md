# constructii-legislatie-ro site

Minimal static Next.js website for the public `constructii-legislatie-ro` repository.

## Local development

```sh
cd site
npm install
npm run dev
```

## Static build

```sh
cd site
npm run build
```

The site uses Next.js App Router and `output: 'export'`.

## Vercel

Deploy with Vercel using `site/` as the project root directory.

No database, authentication, chatbot, or runtime legal processing is required.
