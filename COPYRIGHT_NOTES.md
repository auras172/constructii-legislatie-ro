## Imported official text

Some files in this repository contain text imported from official Romanian public sources (Portal Legislativ, Monitorul Oficial, and relevant authority pages).

For each such file:

- the import source, URL, and access date are recorded in `import-log/`
- the rights note for that specific act is recorded in `metadata/acts/<slug>.json` under `rights_note`

Official legislative text is reproduced here as a public reference from official government sources. This repository is not an official government source. Official sources remain authoritative.

## What is not included

The repository contribution rules (see `CONTRIBUTING.md`) prohibit importing:

- standards, technical norms, or normative content where reuse rights are unclear
- third-party annotations, legal commentary, or paid database content
- content from non-official sources

If reuse rights for a specific act are uncertain, the repository tracks metadata and a source link only, without importing the full text.
