## Dovezi extrase din PDF

PDF-ul oficial ISCIR este pachetul Monitorului Oficial nr. 385 bis și include Ordinul nr. 663/2010 plus anexele PT C 1, C 4, C 6, C 7, C 8, C 9 și C 10.

Elementele verificate prin extracție locală controlată:

- pagina 1: sumarul Monitorului Oficial identifică anexele nr. 1-7 la Ordinul nr. 663/2010 și include PT C 4-2010;
- pagina 2: preambulul ordinului citează explicit Legea nr. 64/2008;
- pagina 2: Art. 2 aprobă PT C 4-2010 și indică anexa nr. 2;
- pagina 2: Art. 8 lit. b) abrogă Ordinul nr. 305/2003 pentru PT C 4/1-2003;
- pagina 3: Art. 10 stabilește intrarea în vigoare la 30 de zile de la publicare;
- pagina 118: pagina de titlu a anexei nr. 2 confirmă indicativul PT C 4-2010 și titlul prescripției: `RECIPIENTE METALICE STABILE SUB PRESIUNE`.

## Ordinul 557/2014

Pagina ISCIR listează separat Ordinul ministrului economiei nr. 557/28.04.2014 privind modificarea Ordinului nr. 663/2010.

PDF verificat: https://iscir.ro/wp-content/uploads/2011/03/557-28apr2014.pdf — HTTP 200.

Conținutul ordinului modificator indică modificări la:

- anexa nr. 2 articolul 116 lit. h);
- anexa nr. 5 articolul 104 lit. h).

Spre deosebire de PT C 1-2010, PT C 4-2010 este afectată direct de Ordinul 557/2014. Această intrare rămâne metadata-only și nu reproduce text consolidat; amendamentul este documentat în `rights_note`.

## Decizie de import

Metadata-only. PT C 4-2010 reglementează recipientele metalice stabile sub presiune și este relevantă pentru siguranța instalațiilor sub presiune urmărite în domeniul ISCIR.

Textul integral nu este importat pentru a păstra scope-ul restrâns și pentru că documentul oficial este un PDF combinat de 496 pagini, care acoperă șapte prescripții tehnice distincte.
