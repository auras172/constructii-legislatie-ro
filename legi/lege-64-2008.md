Legea nr. 64/2008 este importată ca metadata-only deoarece este baza legală directă pentru prescripții tehnice ISCIR deja urmărite în repo, în special PT CR 4-2009 și PT CR 8-2009.

Full-text importul este amânat pentru un PR separat: actul are republicări, formă consolidată la 13.12.2023 și anexe, iar scopul acestui PR este doar completarea provenienței și a grafului.
